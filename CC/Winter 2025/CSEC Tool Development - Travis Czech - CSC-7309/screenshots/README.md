# Screenshots — Evidence Images

## Evidence Strategy

This portfolio uses **embedded terminal output transcripts** rather than screenshot images. This approach is:
- **More verifiable** — text output can be diffed and reproduced
- **More accessible** — renders in any markdown viewer without image loading
- **Version-controlled** — changes are visible in git history

See [EVIDENCE_INDEX.md](../EVIDENCE_INDEX.md#screenshots--compilation-evidence) for the full cross-reference of embedded execution evidence.

## Naming Conventions (for future image captures)

| Pattern | Example | Use When |
|---|---|---|
| `wkNN_<topic>_<index>.png` | `wk04_hangman_1.png` | Tied to a specific week / topic |
| `ScreenshotN_<ShortDesc>.png` | `Screenshot1_CargoBuild.png` | Generic evidence snapshots |

## Tracking

Files in this folder are tracked via **Git LFS** (see `.gitattributes` at repo root).
